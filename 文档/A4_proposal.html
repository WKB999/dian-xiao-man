关键洞察: 学区商圈是典型的"高客流、低数字化"场景—— 客流密度堪比商业区，数字化程度仅为商业区的28%（资料来源: 艾瑞2025校外教育零售报告）。

🔥三大独特痛点

(1) 周期性剧烈
开学季销量是寒假4倍，但90%店主"靠直觉备货"，浪费严重。
(2) 客流"不接住"
开学季流量爆发3天就跌，无营销工具承接转化，月活流失60%。
(3) 全能型店主
3-5人小店同时承担选品/备货/营销/客服/账务5项职能，无暇使用复杂工具。
"暑假一过，奶茶销量跌掉一半。我哪儿知道为什么。"
—— 东北大学南门 奶茶店 · 张老板 · 访谈 2026.6.15

🤖商户端: 4个AI工具

📅 模块A · 学区日历
自定义时间节点（开学/考试/军训/毕业等），AI自动推送营销建议
例: 期末考试周 → "夜宵品增加50%，下午茶减少30%"
✍️ 模块B · AI营销文案
选择店铺真实产品 → 自动生成朋友圈/小红书/群发3套文案
例: 选"珍珠奶茶"+期末周 → 生成3条针对性文案+海报
💡 模块C · AI智能定价
基于商户录入的真实产品列表 + 学区周期动态给出定价建议
例: 珍珠奶茶 ¥15.9→¥12.9（期末客流-35%，降价刺激转化）
💬 模块D · AI客服
挂微信/美团，AI自动应答"几点关门""有折扣吗""能外卖吗"
例: 凌晨问 → "在的！营业到23:30~新生凭学生证9折 🎓"

🎓学生端: 发现优惠 + 下单 + 赚佣金

🔥 发现附近优惠
按距离展示周边商户实时活动（自取更便宜）
例: "老张的茶·买一送一 · 120m · 到店自取¥7.95起"
🛒 一键下单
支持到店自取 / 配送到寝室两种模式
例: 选"自取"→ 出示订单码即可取餐(免配送费)
📤 分享赚佣金
分享活动给室友/同学，成交后赚 3%~8% 佣金
例: 分享奶茶买一送一给室友 → 3人下单=赚¥2.4
⚙️ 商户设置
选商户类型(6种) → 录入产品 → 设主推 → AI适配
例: 选"奶茶店"→录入8款饮品→AI生成对应文案/定价

商业判断 → 免费切入+订阅+分成

💰三层商业模型

① 免费层(引流): 4个AI模块基础版免费 → 快速获客
② 订阅层(核心): 49元/月专业版 → 解锁高级AI(智能定价/日历预警/多店管理)
③ 分成层(增量): 店家在平台发起的学生优惠券交易5%分成

目标: 1年内1万付费店=49万/月=588万ARR（对标: 美团闪店长起步同样量级）
✓ 沈阳试点: 东北大学/沈师/辽大 3校周边50店先行验证

⚖️与美团的核心差异化

能力美团/外卖平台店小满
外卖配送核心能力不自己做（自取+轻配送）
学区周期日历无有（自定义节点）
AI营销文案生成无有（基于产品自动生成）
AI智能定价建议无有（基于产品+周期）
AI客服自动回复无有（微信/美团接入）
学生分享赚佣金无3%~8%佣金激励
商户自助发布优惠需平台审核即时发布，1分钟上线
定位差异: 美团=全品类外卖平台(重配送)；店小满=学校周边促销活动平台+AI营销工具(轻交付、强运营)

护城河 → 三重壁垒

🏗️ 数据护城河
沉淀学区周期数据(每个学校的开学季/考试季/毕业季客流规律) → 数据越多预测越准 → 竞争对手需要2年以上才能复现。
🌐 渠道护城河
锁定每个学校周边头部5-10家店 → 形成该学区事实标准 → 一旦学校商圈成熟，竞品进入门槛极高。
🧠 模型护城河
用5000+店训练出"学区场景"专有模型 → 比通用GPT更懂"学生话术"和"周期备货" → 每次只沉淀1所学校，无法被快速复制。
7

风险与对策

(1) 付费意愿低
小店月流水3-10万，对订阅付费敏感。
→ 免费层长期免费;分成层先抽取学生券增量。
(2) 地推成本高
218万家店需要BD覆盖。
→ 学生兼职单店200元奖励；先用华东2省锚定200店。
(3) 大厂进入
美团/抖音可能下场。
→ 抢占"学区"心智;数据护城河2年;店小满专有模型不可复制。
(4) 学校节奏差异
不同学校节奏不同，模型不可一刀切。
→ 每校独立校历;走"1学校1模型"路径而非全国1套模型。
